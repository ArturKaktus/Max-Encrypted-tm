# Max-Encrypted-tm

Общайся и шифруйся со своими друзьями в мессенжере Max

## Как пользоваться ⚙️
1. Договорись с другом над секретным словом.
2. Оба нажмите сверху на шестеренку и впишите его, нажмите на сохранить
3. Напиши текст, но не отправляй! сначала зашифруй его с помощью кнопки справа
4. Текст изменился на 🔒[A1B2C3D4] и его отправляй
5. Когда получишь такое же зашифрованное сообщение - в том же облачке нажми на кнопку для дешифровки
6. Вы великолепны

## Как установить 💅
1. Установите расширение [Tampermonkey](https://www.tampermonkey.net/) для запуска пользовательских скриптов
2. Установите скрипт, перейдя по [этой ссылке](https://github.com/ArturKaktus/Okko-logo-remover-tm/raw/refs/heads/main/userscript/max-encrypted.user.js). _(либо скачайте `max-encrypted.user.js` из папки `userscript` и установите вручную)_
